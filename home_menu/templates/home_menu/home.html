This web application first of all allows you to store and access ERP data from experiments run at the Faculty of Physics of the University of Warsaw. The data must have a specific EEGLAB format.

The user can firstly upload the data to the server where it is stored. Secondly, it is possible to download the raw data files. However, the main functionality of this application is preprocessing of EEG signals.

EEG Builder provides an extensive interface for the configuration of the settings according to which the experiment data will been preprocessing. In order to obtain a pickle file for downloading and further analysis at the end.

So far, the Department of Biomedical Physics has managed to collect data from 12 experiments involving 36 subjects who responded to 800 stimuli of displaying words on a computer screen. The studies were conducted in three different variants:
  • Lexical decision tasks (LTD)
  • Emotional decision task (ETD)
  • Stroop test
Words are assigned features, for example[1]:
  • valence (positive/negative)
  • origin of the emotion ("from the heart"/"from the mind")
  • meaning (personal/subjective)
  • arousal
  • pseudo-words
[1] - Imbir, K. (2016d). Affective norms for 718 polish short texts (ANPST): dataset with affective ratings for valence, arousal, dominance, origin, subjective significance and source dimensions. Front. Psychol. 7
